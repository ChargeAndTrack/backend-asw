# Ubiquitous language

- **User**: a registered person who logins to use the system;
- **Role**: the role of the user (base user or administrator);
- **Map**: the considered area's representation in which there are multiple charging stations;
- **Place**: a single point in the map (e.g. charging station);
- **Charging station**: a device used to charge electric cars;
- **Electric car**: a car that needs its battery to be charged periodically.
